9 - git branch --> to see number of branches , *refers as the current branch 

10 - git branch -a  --> to see all the local and remote branches 

11 - git branch branchname --> create a branch 

12 - git checkout branchname --> go to that branch , if you have some changes that are not commited then they also goes to that branch . 

13 - git checkout -b branchname --> create a new branch and refers to the new branch this will take all the changes whether in the working directory or not it takes all the data into a new branch which you make.
